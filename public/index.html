📥 Thêm câu hỏi nhanh qua file JSON

Bạn có thể mở file sample_questions.json trong thư mục c:\Quiz, chỉnh sửa nội dung câu hỏi, rồi import vào chủ đề bất kỳ.

Cấu trúc file JSON:

[
  {
    "type": "multiple_choice",
    "content": "What is 'cat'?",
    "option_a": "Con mèo",
    "option_b": "Con chó",
    "option_c": "Con gà",
    "option_d": "Con vịt",
    "correct_answer": "A",
    "explanation": "Cat = con mèo",
    "example_sentence": "The cat is sleeping."
  },
  {
    "type": "essay",
    "content": "Translate: I love you",
    "correct_answer": "Tôi yêu bạn",
    "explanation": "love = yêu",
    "example_sentence": "I love my family."
  }
]
